Frequently asked questions

Where is my data stored?

Lists, scores, history, language, and theme are stored only on your device. Ten Note has no account, Firebase database, advertising, analytics, or tracking service.

How do I move data to another phone?

Open Settings, enter a backup password, and create an encrypted backup. Move the .tennote file to a location you control. On the new phone, enter the same password and choose Merge backup or Replace with backup.

What if I forget the backup password?

Ten Note does not store the password, so an encrypted backup cannot be recovered without it. Create a new backup with a password you can keep safely.

How do I delete my information?

Delete items or lists individually, or use Settings → Delete all local data. Exported backup files must be deleted separately.

Is Ten Note a medical app?

No. Ten Note is a general personal tracking tool and does not provide medical advice, diagnosis, or treatment.
